# ATM Simulation System

A menu-driven ATM simulation program with PIN verification.

## Features
- PIN authentication
- Balance inquiry
- Deposit money
- Withdraw money
- Transaction messages

## Technology
- C Programming
- VS Code
- Git & GitHub

- Practiced conditional statements and loops
- Implemented real-life ATM logic
- Improved problem-solving skills in C
